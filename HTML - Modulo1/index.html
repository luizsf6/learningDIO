Tag H - Títulos e Parágrafos

Este tipo de tag (varia de H1 até H6) é utilizada para realizar marcações de parágrafos como se fossem títulos de diferentes níveis de importância, onde H1 é o mais importante e H6 o menos importante.

Tag P - Corpo textual

Uma tag P é usada para se inserir um corpo em um texto, como se fosse um parágrafo textual. Esse texto, por exemplo, é feito por uma tag P.

Tag Mark - Marcação

A tag <mark> é utilizada para destacar uma parte específica de um texto, fazendo com que ela se sobressaia como se estivesse marcada com um marcador de texto.

Tag Small - Texto Pequeno

A tag <small> reduz o tamanho do texto, usada para exibir textos com menor relevância ou informações complementares.

Exemplo de texto em small.

Tag I - Itálico

A tag <i> é utilizada para colocar um texto em itálico, como este: Exemplo de texto em itálico.

Tag U - Sublinhado

A tag <u> sublinha o texto. É usada para destacar informações específicas. Exemplo de texto sublinhado.

Tag Strong - Ênfase Forte

A tag <strong> é usada para dar ênfase a um texto, exibindo-o em negrito e sugerindo maior importância. Texto em negrito

Tag's ol, ul e li - Listas

As tags <ul>, <ol> e <li> são usadas para criar listas. A tag <ul> cria uma lista não ordenada (com marcadores), enquanto <ol> cria uma lista ordenada (numerada). Cada item de lista é representado pela tag <li>.

Tag A - Âncoras e Links

A tag <a> é utilizada para criar links de ancoragem, tanto para outras páginas como para seções específicas dentro da própria página. Exemplo: Ir para Tag HR

Tag HR - Linha Horizontal

A tag <hr> é usada para criar uma linha horizontal, normalmente para separar seções de conteúdo.

Tag's Sub e Sup - Subscrito e Sobrescrito

As tags <sub> e <sup> são usadas para textos subscritos e sobrescritos, respectivamente. Exemplo: CO2 e E=mc2.

Tag Blockquote - Citação em Bloco

A tag <blockquote> é usada para marcar uma citação longa. Exemplo:

"HTML (HyperText Markup Language) é a linguagem de marcação padrão para a criação de páginas da Web."

Tag Font - Alterar Fonte

A tag <font> é utilizada para alterar a cor, o tamanho e a família da fonte. Entretanto, esta tag está descontinuada e não deve ser usada em projetos modernos.

Exemplo de texto com a tag font.

Tag Del - Texto Deletado

A tag <del> representa um texto que foi removido ou alterado. Exemplo: Este texto foi removido.

Tag Abbr - Abreviação

A tag <abbr> é usada para indicar abreviações. Exemplo: ONU.
